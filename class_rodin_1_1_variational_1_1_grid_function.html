template<class FES>
Rodin::Variational::GridFunction class

Represents a grid function which belongs to some finite element space.

Template parameters
FES Finite element collection to which the function belongs.

Contents

  • Reference
    • Public static functions
    • Constructors, destructors, conversion operators

Note

Note that the FES template parameter is typically inferred when initializing the grid function, hence it is not necessary to make it explicit.
