template<class FEC>
Rodin::Variational::Problem class

Represents a variational problem to be solved.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

The problem may be specified via the overloaded operator Variational::Problem::operator=(const FormLanguage::ProblemBody&).

The problem may then be solved by utilizing any derived instance Solver class in the Rodin::Solver namespace.
