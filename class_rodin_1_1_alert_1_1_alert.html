Rodin::Alert::Alert class

Base class for objects which represents alerts.

Contents

  • Reference
    • Derived classes
    • Constructors, destructors, conversion operators
    • Public functions

Represents an Alert which can be raised and will have visible effects. These effects may include but are not limited to:

  • Outputting to stdout or stderr
  • Asking user for input through stdin
  • Writing to a log file
  • Aborting the program
  • Any combination of the above.

Every Alert has a message containing its description or reason for being raised, which may be obtained by the what() method. Furthermore, an Alert may be raised via its raise() method.
