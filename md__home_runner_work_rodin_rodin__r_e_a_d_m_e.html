Requirements

  • CMake 3.12.0+

Building the project

git clone --recursive https://github.com/carlos-brito-pacheco/rodin
cd rodin
mkdir build && cd build
cmake ..
make -j4

CMake Options

OptionDescription
RODIN_BUILD_EXAMPLESBuilds the examples in the examples/ directory.
RODIN_BUILD_DOCBuilds the documentation using Doxygen
RODIN_USE_MCSSBuilds the documentation using Doxygen and m.css
