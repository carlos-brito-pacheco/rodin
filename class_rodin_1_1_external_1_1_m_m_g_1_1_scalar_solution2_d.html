Rodin::External::MMG::ScalarSolution2D class

Scalar solution supported on a 2D mesh.

An object of type ScalarSolution2D represents a function

whose known values are given on vertices of some mesh .

ScalarSolution2D& Rodin::External::MMG::ScalarSolution2D::setMesh(Mesh2D& mesh)

Sets the associated mesh.

Parameters
mesh in Reference to mesh.
Returns Reference to self (for method chaining).

Note

The method does not check to see if the mesh is compatible with the current data in the solution. In general, it is up to the user to ensure that the number of points are the same, keep track of the modifications to the underlying mesh, etc.

Note
