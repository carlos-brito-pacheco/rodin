Rodin::Geometry::Point class

Represents a spatial point which belongs to some element of a mesh.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

A Point represents the physical coordinates (as opposed to reference coordinates) of a point on the mesh. This class differs from a Vertex in the sense that a Vertex is a node of some element of a Mesh. In contrast, a Point represents any coordinates contained in the Mesh.
