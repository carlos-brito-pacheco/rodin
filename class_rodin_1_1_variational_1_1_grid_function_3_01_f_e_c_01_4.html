template<class FEC>
Rodin::Variational::GridFunction<FEC> class

Represents a grid function which belongs to some finite element space.

Template parameters
FEC Finite element collection to which the function belongs.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

Note

Note that the FEC template parameter is typically inferred when initializing the grid function, hence it is not necessary to make it explicit.
